# Boolflix

A Netflix-style movie browser built with Vue.js, consuming TheMovieDB API for live movie and TV series data.

## Features
- Search movies and TV series by keyword
- Posters, ratings, language and overview from live API
- Component-based architecture with parent-child props

## Tech
- Vue.js
- Axios for API calls
- HTML5 / CSS3
- TheMovieDB API

## Setup
1. Clone the repo
2. `npm install`
3. Add your TheMovieDB API key in the config
4. `npm run serve`
